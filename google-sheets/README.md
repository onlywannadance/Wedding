# Запись анкеты в Google Таблицу

## 1. Таблица

Создайте Google Таблицу. В **первой строке** задайте заголовки (именно в таком порядке):

| A      | B        | C     | D            | E        |
|--------|----------|-------|--------------|----------|
| Имя    | Партнёр  | Дети  | Присутствие  | Напитки  |

Скопируйте **ID таблицы** из ссылки в браузере:
`https://docs.google.com/spreadsheets/d/ **ВОТ_ЭТОТ_ID** /edit`

## 2. Apps Script

1. Откройте **именно ту таблицу**, в которую должны писаться анкеты.
2. В ней: **Расширения** → **Apps Script** (скрипт должен быть привязан к этой таблице, не создавайте отдельный проект на script.google.com).
3. Удалите весь код в файле `Code.gs` и вставьте содержимое файла `Code.gs` из папки `google-sheets` этого проекта.
4. Сохраните (Ctrl+S). Подставлять ID таблицы не нужно — скрипт пишет в ту таблицу, из которой открыт.

## 3. Развёртывание

1. Нажмите **Развернуть** → **Новое развёртывание**.
2. Тип: **Веб-приложение**.
3. **Выполнять от имени**: я (ваш аккаунт).
4. **У кого есть доступ**: **Все**.
5. Нажмите **Развернуть**, при первом запросе разрешите доступ.
6. Скопируйте **URL веб-приложения** (окно «Веб-приложение развёрнуто»).

## 4. Сайт

В файле **js/main.js** вставьте скопированный URL в переменную `SHEETS_WEB_APP_URL`:

```js
var SHEETS_WEB_APP_URL = 'https://script.google.com/macros/s/.../exec';
```

После этого при отправке анкеты данные будут записываться в таблицу: **одна строка на каждого гостя** (имя, партнёр, дети, присутствие, напитки).

### Локальная разработка (Live Server)

При открытии сайта с `http://127.0.0.1` или `http://localhost` браузер блокирует запросы к Google из‑за CORS. Чтобы форма работала локально:

1. В папке проекта запустите прокси (достаточно одного варианта):
   - **PowerShell (Windows, без установки):** `powershell -ExecutionPolicy Bypass -File proxy-server.ps1`
   - **Python:** `python proxy-server.py`
   - **Node.js:** `node proxy-server.js`
2. Откройте сайт через Live Server и отправляйте анкету — запросы пойдут через прокси на порт 3456.
3. На выложенном в интернет сайте прокси не нужен: форма обращается к Google напрямую.

Запуск local сервера:
powershell -ExecutionPolicy Bypass -File proxy-server.ps1
